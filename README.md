# 基于大数据反电信诈骗管理系统 python1181

## 项目概述

本项目是一款基于大数据技术的反电信诈骗管理系统，采用B/S架构，利用Python语言进行开发，并结合MySQL数据库进行数据存储。该系统致力于通过文本分析、文本管理等功能，提升对电信诈骗行为的识别和管理效率。

## 技术栈

- **后端语言**: Python
- **框架**: Django
- **数据库**: MySQL
- **编译器**: PyCharm
- **Python版本**: 3.7

## 功能模块

- 文本分析
- 文本管理
- 修改密码
- 个人信息
- 用户管理

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- Python 3.7+
- MySQL 数据库环境

## 部署步骤

1. 确保Python环境为3.7版本。
2. 安装所需依赖库。
3. 配置MySQL数据库。
4. 运行系统。
